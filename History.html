TUTORIAL

Copiar 🏔️ Texto

DESCRIPTION

Dwarf Vault is an extension that allows you to save text from web pages quickly and easily. 📚✨ With just one click on the text you want, you can save the text, URL, and favicon of any page into a custom database that you create. 🗂️🌍 You can have as many databases as you need and access them quickly. 🚀 This way, saving and finding your favorite texts will always be easy and organized! 📑🔒

Funciones

1- Create Your First Database 🏗️:

- Open extension → Go to "FORGE 🔨" tab → Click "FORGE 🔨 - Create New Database" → Add a name → DONE! ✅

2- Save Your First Data 📥:

- Select text → Right-click → "The Dwarfs Vault" → 📥 Save to Vault 🏰 → Select your database → DONE! ✅

3- View Saved Data 👀:

- Right-click → "The Dwarfs Vault" → Select your database → Click on the data → RUNES ⚒️ opens → DONE! ✅

4- Copy Text to Clipboard 📋:

- Right-click → "The Dwarfs Vault" → Select your database → Click on the data → RUNES ⚒️ opens → Click "Copy Text" → DONE! ✅

Note: If it doesn’t open, interact with the extension first (open "FORGE 🔨" or click on it).

5- Open Saved URL from Text 🌐:

- Right-click → "The Dwarfs Vault" → Select your database → Click on the data → RUNES ⚒️ opens → Click "Open Link" → DONE! ✅

Note: If it doesn’t open, interact with the extension first (open "FORGE 🔨" or click on it).
