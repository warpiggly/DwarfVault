How to use 🏔️ DwarfVault

WHAT IS DWARFVAULT?

DwarfVault is a browser extension that lets you save text from any web page with one right-click. Build your own libraries of notes, links, snippets or quotes and get them back in seconds — all stored locally on your computer. 🔒

KEY CONCEPTS

🏛️ Vault — a main database. Think of it as a folder (e.g. "Work", "Recipes").

📦 Chest — a sub-database inside a Vault. Useful to split a Vault into topics (e.g. Work → Emails, Work → Quotes).

📜 Entry — a single saved item: text + URL + favicon of the page it came from.

QUICK START (3 STEPS)

1 — Create your first Vault 🏗️

Open the extension → expand FORGE 🔨 → click ⛏️ NEW VAULT → type a name → done! ✅

2 — Save text from a web page 📥

Select any text on a page → right-click → 📥 Save to Vault 🏰 → pick your Vault → done! ✅ The text, URL and favicon are stored together.

3 — View what you saved 👀

Option A: open the extension → expand RELICS 🏰 → search or scroll through your entries.

Option B: right-click anywhere → The Dwarf's Vault → 📂 Vault → pick an entry → the RUNES ⚔️ panel opens at the top of the popup with its text and link.

THE FORGE 🔨 — MANAGE YOUR VAULTS

🏛️ Dwarf Chamber

The dropdown at the top. Select which Vault (or Chest) you want to work on. Every action below applies to the one selected here.

Management buttons

⛏️ NEW VAULT — create a new main database.

🔨 NEW CHEST — create a sub-database inside the selected Vault.

🛠️ REFORGE — rename the selected Vault or Chest.

💀 DESTROY — delete it forever. Be careful! ⚠️

Trade — Import & Export

📤 EXPORT CSV / 📥 IMPORT CSV — back up or load a single Vault/Chest as a spreadsheet file.

📦 EXPORT Full Vault / 🗃️ IMPORT Full Vault — back up or load a Vault with all its Chests at once (JSON file).

QUICK ACCESS — 1-CLICK MAGIC ⚡

Quick Access puts your favorite Vault items into the right-click menu, so you don't have to open the popup every time.

⭐ Favorites — copy with one click

Pick a Vault in the ⭐ FAVORITES dropdown. Now, on any page, right-click → The Dwarf's Vault → ⭐ Favorites → click an item → the text is copied to your clipboard instantly. 📋

🔗 Links — open URLs with one click

Pick a Vault in the 🔗 LINKS dropdown. Then right-click → The Dwarf's Vault → 🔗 Links → click an item → the URL opens in a new tab. 🌍

⛓️ LINK — Sync Both

Click this button to use the same Vault for Favorites and Links. Handy when one Vault has both text and links you use often.

⚙️ Change the active Vault from the right-click menu

Right-click → The Dwarf's Vault → ⚙️ Set Active Vault → pick any Vault/Chest. The checkmark (✓) shows the current one. Choose (None) to disable Quick Access.

RELICS 🏰 — SEARCH & EDIT

Inside the popup, expand RELICS 🏰 to see every entry of the selected Vault. Use the search bar to filter by index or text. From there you can copy, edit or delete each entry. 🔍

RUNES ⚔️ — THE ENTRY PANEL

When you open an entry from the right-click menu (📂 Vault → entry), the top panel of the popup — RUNES ⚔️ — shows its index, favicon, text and URL. Use Copy Text to copy it exactly as saved (line breaks included).

💎 GLYPH VAULT

At the bottom of the FORGE section. Click to open the emoji grid and click any emoji to copy it to your clipboard. ✨

TIPS & FAQ

The right-click menu doesn't open my popup — why?

Browsers require a first interaction with the extension in each session. Open the popup once (click the icon or expand FORGE), then the right-click → popup flow will work normally.

Where is my data stored?

Everything lives locally in your browser (IndexedDB). Nothing is sent to any server. If you change browsers or clear site data, your Vaults may be lost — use EXPORT to keep backups.

Can I have many Vaults?

Yes — as many as you need, each with its own Chests. The menu shows a count of sub-DBs in brackets, e.g. "Work [3 sub-DB]".

Keyboard shortcut

A keyboard command can open the extension popup. Set or change it at chrome://extensions/shortcuts. ⌨️
